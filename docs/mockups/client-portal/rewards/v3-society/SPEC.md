# Rewards Section Update – Body Shaper System Society™

**Status:** Approved direction (per Emmy, supersedes earlier `rewards/` v1/v2 mockups)

We are redesigning the Rewards section to create a premium members club experience. This should NOT feel like a generic loyalty program. It should feel like an exclusive wellness membership integrated into the Blueprint ecosystem.

The Rewards section should now be divided into 4 internal tabs.

---

## 1. Overview

This is the main dashboard.

Include:
- Welcome message: **Welcome to The Body Shaper System Society™**
- Current Points
- Membership Level
- Progress bar to the next reward
- "Next Reward to Unlock" card
- Recent Activity
- Ways to Earn Points
- Beautiful editorial cards
- Luxury dashboard aesthetic

This page should immediately communicate exclusivity and motivate members to keep earning points.

Mockup: `overview.png`

---

## 2. Unlock Experiences

This page contains all redeemable rewards.

Organize everything into elegant categories such as:
- Treatments
- Beauty
- Wellness
- Experiences
- Partners
- Holistic
- VIP Experiences

Each reward should appear as a premium editorial card.

Locked rewards should display elegant lock icons with messages like: **"Unlock at 2,500 Points"**

Some experiences remain blurred until unlocked.

Think **Apple + Soho House + American Express Platinum.**

Mockup: `unlock-experiences.png`

---

## 3. Secret Missions

This is the gamification area.

Create beautiful mission cards such as:
- Weekly Mission
- Zodiac Challenge
- Secret Challenge Drop
- Social Media Mission
- Referral Mission
- Birthday Mission
- Seasonal Challenge

Each mission displays:
- Points earned
- Difficulty
- Progress
- Claim button
- Countdown when applicable

This section should encourage clients to interact with the brand regularly.

Mockup: `secret-missions.png`

---

## 4. Privileges

This is **NOT** another rewards catalog.

This page showcases exclusive membership benefits.

Examples:
- Beauty Partner Benefits
- Wellness Partner Discounts
- Private Events
- VIP Invitations
- Members Only Experiences
- Early Access to New Protocols
- Birthday Gifts
- Exclusive Launches
- Priority Booking

This should feel like entering an exclusive private club.

Many benefits can remain locked until reaching higher membership levels.

Mockup: `privileges.png`

---

## General Design Direction

Keep the exact same editorial language used throughout the Blueprint.

Avoid anything that looks like Shopify, Square, or a generic rewards app.

Think:
- Luxury wellness club
- Editorial magazine
- Premium membership
- Soft gradients
- Rounded premium cards
- Clean typography
- Elegant spacing
- Gold, mocha, nude and burgundy accents
- Minimal icons
- High-end UI

Everything should feel consistent with the Body Blueprint experience.

**Title remains:** Rewards
**Subtitle:** The Body Shaper System Society™
