# Design System — Card Backgrounds (LAST VERSION)

**These are the current/final full-bleed card background textures** — use these over anything in the parent `design-system/` folder's earlier tiled sheets when picking an actual background image for a card.

## Received
- `olive-marble-gold-waves.png` — sage/olive marble with the signature thin gold double-wave line motif in two corners
- `taupe-marble-gold-waves.png` — warm taupe/greige marble with the same gold double-wave motif
- `bronze-metal-compass-star.png` — brushed bronze metal with an engraved compass-ring + 8-point star (BSS compass mark), diagonal light sheen
- `bronze-metal-plain.png` — plain brushed bronze/copper metal, diagonal soft light bands, no engraving (pairs with the compass-star version above for cards that don't need the mark)
- `linen-fabric.png` — sand-toned woven linen texture, flat/matte
- `ribbed-glass-cream.png` — cream/champagne ribbed (fluted) glass texture
- `cream-marble.png` — clean ivory/cream marble, subtle veining, no overlay

## Notes
Full-bleed, no text/labels — ready to use directly as a card background image (as opposed to the earlier tiled swatch sheets in `../background-textures.png`, `../rewards-card-backgrounds.png`, etc., which were reference sheets showing many options at once). When in doubt about which texture to use for a given card, use one from this folder.
